# RTL Skills

Backup of reusable Codex RTL design skills.

## Contents

- `skills/rtl-module-design/SKILL.md`: main RTL module design skill.
- `skills/rtl-module-design/references/`: reusable reference pages for module templates, handshake paths, FIFOs, arbiters, and FSMs.

## Source

Recommended local install path:

```text
$CODEX_HOME/skills/rtl-module-design
```

This repository intentionally stores only skill instructions and reference notes. Do not add Codex auth files, memory databases, logs, tokens, or user secrets.
